Frequently Asked Questions

Find the answers for the most frequently asked questions below

What is pAIges and how does it work?

pAIges is an intelligent data extraction engine based on Machine Learning algorithms that interpret and identify OCR extracted data and make them available in an easily usable format for further processing.

What are the document types trained in pAIges?

Various kinds of documents can currently be processed through pAIges. These include Invoices, KYC documents, travel documents, PoDs and Sales Orders. Many of these are regions specific and pAIges is able to process those accordingly. If you have a different document type you can write to us at {{ADMIN_EMAILID}} and we will get back to you.

Can pAIges extract all the fields and data from an Invoice?

pAIges is currently trained to extract 35 key fields from an Invoice.

What if a field is not extracted correctly?

This can happen because of the quality of the scanned document that you are using. If you feel that is not the case then we request you to share the field information along with document details at {{ADMIN_EMAILID}}

Can a new field be added?

Yes, we can add new fields that you might require. Please share the field details along with the relevant document at {{ADMIN_EMAILID}}

Can pAIges integrate with any other tools?

Yes, there are Rest APIs available that can be accessed by any application, in the Licensed version.

Can the output format be customized?

Currently, pAIges can provides output in Excel and JSON formats. It can also be customized to provide outputs in other formats if required.

How secure is my data in pAIges?

Your data is completely secure and is encrypted both at motion and at rest.

Can pAIges be trained on new document type?

Yes, please let us know what other document types you need to extract data from, along with details of the data to be extracted at {{ADMIN_EMAILID}}
